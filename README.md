A web app made with **React**, **Tailwind CSS** and Movie DB API that enables searching for movies, showing movies including their ratings and date of release as well as trending movies according to user's search history. 

Based on following the tutorial  [React JS 19 Full Course 2025 | Build an App and Master React in 2 Hours](https://www.youtube.com/watch?v=dCLhUialKPQ&t=545s).
